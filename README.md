# Zipcode Lookup

Aplicação para demonstrar a capacidade de Progressive Web Apps.

## Running locally

```sh
# Install npm dependencies
$ npm i

# Run development server for Vite
$ npm run dev
```

## Build project

```sh
# Install npm dependencies
$ npm i

# Run development server for Vite
$ npm run build

# Deploy dist folder
```
